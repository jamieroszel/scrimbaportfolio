Image Endless

Store images you find online

Photo by freddie marriage on Unsplash

Image Endless is a web app built with Node.js, Express, and MongoDB that allows authenticated and authorized users to store and display images found online.

The page was built using Bulma, "a free, open source framework that provides ready-to-use frontend components that you can easily combine to build responsive web interfaces" and deployed using Heroku.
